glas.sh

A native visionOS SSH terminal app with a glass-first UI and multi-window workflow.

View on GitHub Sponsor

Current implementation

  • Native SSH session flow using Citadel with PTY support
  • SwiftTerm-backed rendering for ANSI and truecolor terminal output
  • Multi-window layout for connections, sessions, previews, and settings
  • VisionOS ornaments for server context, tools, and session status

Project

Built in Swift for visionOS. Open glas.sh.xcodeproj, choose the glas.sh scheme, select a visionOS simulator, and run.
