Native stack

Swift + SwiftUI + SwiftTerm + Citadel on visionOS.

Multi-window

Connections, terminals, settings, preview, and forwarding surfaces.

Operational clarity

Host key verification modes, SSH key auth, and explicit trust diagnostics.

Feature set

Built for real shell work, not toy demos.

Interactive PTY sessions

Sessions run as interactive terminals with resize propagation, not one-off command execution.

Search + session controls

Footer-first terminal controls provide quick search, clear, preview, and in-window settings.

Session-level overrides

Tune per-session appearance and behavior without disturbing your global defaults.

Host trust workflow

Ask/Strict/Insecure verification modes and explicit fingerprint prompts for first-seen or changed keys.

SSH key management

Import RSA/Ed25519 keys, generate Secure Enclave-backed P-256 keys, and assign keys per server.

Config import and mapping

Import OpenSSH config entries and map identity files to stored keys when possible.

Workflow

From connection to command loop in seconds.

  1. 1. Select server

    Filter by favorites, recents, tags, or direct search.

  2. 2. Verify trust

    Review host key fingerprint and choose trust behavior.

  3. 3. Work in-terminal

    Use PTY shell sessions with dynamic resize and terminal-local controls.

Security posture

Direct about what ships now and what is in progress.

  • Live now

    Password auth, SSH key auth (RSA/Ed25519), and Secure Enclave-backed P-256 key generation.

  • Live now

    Host-key verification modes (Ask / Strict / Insecure) with trust prompts and per-server trust storage.

  • In progress

    SSH agent authentication path.

  • In progress

    Active SSH forwarding lifecycle start/stop wiring.
